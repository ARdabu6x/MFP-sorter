# MFP-sorter
This sorter allows you to sort your files by their extensions. You can use the default file sorting structure or create one by yourself.

## How to run
1. Go to the **Releases** tab on the right.
2. Download the `MFP-sorter.exe` file.
3. Run the downloaded file on your computer.

### How to use
1. Drop the downloaded file into the folder you want to sort.
2. Run it.
3. Select your preferred way to sort files:
   1. **Standard (Default)**: Select this option if you want a quick setup without configuration.
   2. **Custom**: Select this option and click the "Create custom hierarchy" button to create your own structure. Enter a Python dictionary, similar to the format shown after clicking the "Show standard hierarchy" button.
4. Other functions:
   1. **Disable files**: If you want to skip certain files, click the "Disable files" button and enter a Python list, for example: `["code.py", "text.txt"]`.
   2. **Fix**: This feature takes all files from top-level directories (even if you put your own folders or files there) and sorts them again.
5. Finally, click the "Submit" button and see your results.

# MFP-sorter
Ten program pozwala na sortowanie plików według ich rozszerzeń. Możesz użyć domyślnej struktury sortowania lub stworzyć własną.

## Jak uruchomić
1. Przejdź do zakładki **Releases** po prawej stronie.
2. Pobierz plik `MFP-sorter.exe`.
3. Uruchom pobrany plik na swoim komputerze.

### Jak używać
1. Przenieś pobrany plik do folderu, który chcesz posortować.
2. Uruchom go.
3. Wybierz preferowany sposób sortowania plików:
   1. **Standardowy (Domyślny)**: Wybierz tę opcję, jeśli chcesz szybko posortować pliki bez dodatkowej konfiguracji.
   2. **Własny**: Wybierz tę opcję i kliknij przycisk „Stwórz własną strukturę”, aby stworzyć własną strukturę. Wpisz słownik Pythona analogicznie do formatu, który pojawia się po kliknięciu przycisku „Pokaż standardową strukturę”.
4. Pozostałe funkcje:
   1. **Wyłącz pliki**: Jeśli chcesz pominąć określone pliki, kliknij przycisk „Disable files” i wpisz listę Pythona, na przykład: `["kod.py", "tekst.txt"]`.
   2. **Napraw foldery**: Ta funkcja pobiera wszystkie pliki z katalogów głównych (nawet jeśli umieścisz tam własne foldery lub pliki) i sortuje je ponownie.
5. Na koniec kliknij przycisk „Potwierdź”, aby zobaczyć wyniki.
